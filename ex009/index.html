Trabalhando com listas

Listas ordenadas

Para criarmos listas numeradas, devemos usar a tag "ol", ordened list, e dentro dela, usar outra tag nos itens que voce deseja fazer a lista, essa é a tag "li" ela é usada para numerar as listas, segue o exemplo:

  1. Acordar
  2. Comer
  3. Rir
  4. almoçar
  5. Dormir

Junto da tag "ol" podemos adicionar a tag "type" e nela colocar o tipo da lista, se colocarmos um "A", a lista vai de "A" a "Z" em maisculo, se pormos "i", vira numeros romanos, e se colocar "1" fica numerada padrao. Pode tambem adicionar a tag "start ex 4" para a lista começar do numero 4 e etc.

Listas não ordenadas

Para adicionarmos listas nao ordenadas devemos fazer o mesmo das listas ordenadas, porem devemos usar a tag "ul". Nela temos os "types" os tipos "circle" "disc" e "square"

  • Salada
  • Kiwi
  • Maisena
  • Ervilha
  • Goiaba
  • Maracujá

Lista de Definições

é usada para definir termos, como se fosse um dicionario, para usarmos ela devemos utilizar abrir a tag "dl" e dentro dela colocar a tag "dt" para adicionarmos o termo, e depois a tag "dd" para definir o termo. Segue o exemplo.

Pdp
Giria usada para falar um "ok"
Obg
Abreviação usada em redes sociais para dizer "obrigado"
