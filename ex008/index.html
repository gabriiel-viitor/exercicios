principais formataçoes

Negrito/Destaque

Para usar o Negrito temos que utilizar o comando "b", porem essa tag nao tem semantica, nao tem signficado, so tem forma

para usar o Negrito semantico, com sentido, devemos utilizar a tag "strong"


para usar um termo em Italico, devemos utilizar a tag "I"

Para utilizar o Italico com semantica, devemos utilizar o comando "EM"

a semantica é o sentido, essas tags sem semanticas nao tem sentido, mas funcionam, agora as com semanticas tem sentido, o "Strong" vem de forte, o negrito pode ser uma parte "forte" da frase, o "EM" significa enfase, que o italico é utilizado para frases com enfase, por isso essa tag é semantica
